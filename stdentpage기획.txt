사이드바 | 메인 영역




사이드바는 "서치사이드바" 역할을 하고 학생 기본 정보를 보여줌. 검색은 단순 이름 검색과 세부 검색으로 이중화되고

-----------------------

----------------------
상세검색

필터

필터링된 원생 목록
----------------------


구조별 필요 기능

기본검색 : 검색 결과는 z-index를 달리하여 기본검색 바로 밑으로 뜨고, 키보드 네비게이션이 가능함. 그리고 결과를 불러오면 (클릭을 하거나 엔터를 치면) 검색 결과창은 사라짐.

학생기본정보는 스켈레톤이 필요함

원생등록 버튼 : sheet의 형태로 사이드바 오른쪽에서 슬라이드되어 나타나야 하고, 높이는 사이드바 영역과 같은 높이를 가짐. z-index도 사이드바와 같은 값을 가짐. 등장시 메인 영역은 어두워져야 함.
원생상세/수정 버튼 : 원생등록 버튼과 동일함

상세검색 : 필터링된 검색의 결과를 보여주는 구조로, 필요한 필터는 나중에 추가할 수 있게 구조화되어야 함. 
검색 초기화 버튼과 원생검색 버튼이 필요하고

필터링된 원생목록은 간단한 2열의 테이블 형식으로 보이면 됨. (클릭하면 세부창으로 이동)



메인 영역은 각 학생의 세부 정보를 보여주는 역할을 하고 

학생 이름
퀵액션
탭 [기본  |  반  |  수납  |  출결  |  상담  |  과제  |  기타] : 기본탭은 각 탭의 내용을 요약해서 보여주는 역할. 다른 탭은 다른 기능들이 개발되면 각각 추가될 예정.

으로 구성됨.
